7. Authentication & Admin System

7.1 JWT-Based Authentication

The planned authentication uses JSON Web Tokens for stateless session management:

Login Flow:
  • Admin enters email/password in login form
  • Frontend sends POST request to /api/auth/login
  • Backend validates credentials against database
  • If valid, generates JWT with payload:
`javascript

{

userId: admin._id,

email: admin.email,

role: admin.role,

exp: Date.now() + 24 * 60 * 60 * 1000 // 24 hours

}

`
  • Frontend stores JWT in localStorage
  • Subsequent requests include JWT in Authorization header:
`

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

` Security Measures:
  • Password Hashing: Bcrypt with 10 salt rounds (industry standard)
  • Token Expiration: 24-hour lifespan (reduces stolen token risk)
  • HTTPS Only: Prevents man-in-the-middle attacks
  • HttpOnly Cookies (Alternative): More secure than localStorage (XSS protection)

Cancelled

` Status Definitions:

8.5 Real-World Rental Workflow Simulation

Day 1 (Booking):
  • Customer submits booking request
  • System creates pending booking
  • Admin receives notification
  • Admin reviews and confirms
  • Customer receives confirmation message
Day 2-14 (Pre-Rental):
  • System sends reminder 24 hours before pickup
  • Admin prepares vehicle (cleaning, fuel, inspection)
  • Customer can modify/cancel (with policy rules)
Day 15 (Pickup):
  • Customer arrives at location
  • Admin marks booking as "active"
  • Car status changes to "unavailable"
  • Inspection checklist completed
Day 15-20 (Rental Period):
  • Car shows as unavailable on website
  • Customer uses vehicle
  • Support available via WhatsApp
Day 20 (Return):
  • Customer returns vehicle
  • Admin inspects for damage
  • Admin marks booking as "completed"
  • Car status changes to "available"
  • Invoice generated (if not prepaid)
Post-Rental:
  • System requests review/rating
  • Data stored for analytics
  • Customer added to mailing list for promotions

This workflow mirrors traditional rental agencies while adding digital efficiency.
